# **App Name**: PowerHub CRS

## Core Features:

- Map-Based Interface: Display interactive map with toggle layers for power, flood, waste, transport and markets.
- Reporting System: Allow users to report issues such as power outages, floods and waste management problems with photo attachment and location auto-fill.
- Service Marketplace: Connect residents with local service providers, farmers and artisans. Display rating, distance, quick actions.
- Transport Guide: Route fare estimator and heatmap overlay for transport activity. Also give fare information for common routes.
- Community Alerts: Reverse-chronological feed of recent reports with map preview. Upvote reports to confirm them.
- Neighborhood Status Tool: The LLM monitors new report submissions and dynamically generate real-time stats related to an area, which are then displayed using "status chips".
- User Dashboard: Personalized dashboard for users to manage reports, favorites, wallet and provider listings.

## Style Guidelines:

- Primary color: Deep green (#0B6B3A), reminiscent of nature and conveys stability and trust.
- Background color: Very light green (#F0FAF4), similar in hue but much lighter, creates a calm and uncluttered background.
- Accent color: Sunrise orange (#FF8A00), provides contrast and energy, drawing attention to calls to action and highlights.
- Headline font: 'Space Grotesk' for headlines and short amounts of body text.
- Body font: 'Inter' for longer text.
- Flat and simple icons representing different services and categories. Use map pins with lightning bolt and leaf for the main logo.
- Map-first layout with a floating action button and bottom tab navigation.
